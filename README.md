# GRChombo Gregory-Laflamme Project

Repository scaffold for a GRChombo-based 5D Gregory-Laflamme black-string project: plans, logs, setup notes, scripts, small results, manifests, and project-specific code placeholders. The long-term goal is gravitational-wave / radiation extraction during the GL transition.

Current milestone: reproduce Pau / Figueras-Andrade-Franca-Gu style 5D black-string horizon diagnostics before adding radiation extraction.

## Repo Map

- `research_plan/`: editable science plans.
- `logs/`: chronological project and Codex logs.
- `docs/`: setup, GRChombo maps, and implementation notes.
- `scripts/`: reproducible run, plot, collection, and cleanup wrappers.
- `code/`: project code placeholders and future patches.
- `results/`: small plots, metadata, summaries, and validation notes.
- `run_manifests/`: structured run records.
