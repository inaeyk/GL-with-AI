# Mac and Codespaces Setup

Single source of truth for Mac/Codespaces notes.

- Apple Silicon Mac: use for editing, Git, Codex, plotting, and post-processing.
- Prebuilt `amd64` GRChombo binaries in Docker may fail on Apple Silicon with `Illegal instruction`.
- Codespaces/cloud VM: useful for quick `x86_64` Docker smoke tests.
- Avoid long interactive runs in Codespaces unless needed.
