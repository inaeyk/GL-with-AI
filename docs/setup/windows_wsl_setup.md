# Windows WSL2 Setup

Single source of truth for Windows setup.

1. Install WSL2 and a Linux distribution.
2. Install Docker Desktop with WSL integration enabled.
3. Clone this repository inside the WSL filesystem.
4. Pull or build the GRChombo Docker image.
5. Run the `BinaryBH` very-cheap smoke test.
6. Store raw outputs in `runs/` or `data/`.
7. Commit only code, docs, scripts, logs, small plots, metadata, and manifests.
